## Вывод на экран

## Вывод c разделителем

```python
a = 5
b = 1
c = 3
print(a,b,c, sep=", ")
# по умолчанию - пробел
```

## Вывод c окончанием

```python

b = 1
c = 3

```

## Ввод строки

```python
name = input("Введите свое имя: ")
print("Здравствуйте", name ,"!")
```

## Ввод целого

```python
a = int(input("Введите целое число: "))
print("Ваше число", a)
```

## Ввод нескольких целых

```python
a, b = map(int,input("Введите два целых числа: ").split())
print("Ваши числа", a, b)
```
